  • filter_dramaHow to get set up
    Just download the source code (or the binaries, if you are on windows!) and run main.py. It may throw an error that you do not have a package installed, so just use 'pip install ' and then the name of the package. It's easy!
  • placeInstalling extensions
    Go to your selected source of mods (webste coming soon) and downlaod one - it should have a '.py' filename. Then, place it in the 'mod' directory. To enable it, go into 'config.json' and add it to the array 'enabledmods'. Now, you are all set!
  • whatshotmaking extensions

    Setting it up

    Create a file in the mod directory with it's file name being whatever you want the mod name to be. Save it with a '.py' extension Go into 'config.json' in the root directory and add your mod name (without the '.py') to the array 'enabledmods'

    API

    Create a function named 'outsource' with one of it's parameters being 'text'. Do whatever you need to do, and when you are finished and need the assistent to say something, make it return the text you want it to say. If you do not want it to say anything, it is best practice to return None.
